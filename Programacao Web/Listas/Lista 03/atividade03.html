  5. Equipamentos para Redes de Computadores
    6.
    Switch
    É um dispositivo utilizado em redes de computadores para reencaminhar pacotes (frames) entre os diversos nós.
    HUB
    É o processo pelo qual se transmite ou difunde determinada informação, tendo, como principal característica, que a mesma informação está sendo enviada para muitos receptores ao mesmo tempo (broadcast).
    Roteador
    É um dispositivo que encaminha pacotes de dados entre redes de computadores, criando um conjunto de redes de sobreposição.
